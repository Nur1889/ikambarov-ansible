Ansible-Role-PHP
=========
This is a simple role which installs PHP 7.0/7.1/7.2/7.3

Requirements
------------
None

Role Variables
--------------
php_version: "70"

Dependencies
------------
None

Example Playbook
----------------
  roles:
    - role: ansible-role-php
      vars:
        php_version: "70"
